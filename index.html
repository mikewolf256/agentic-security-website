Common Questions

Where does this run and what's required?

Deployment: Small service in your VPC, gateway, or as a sidecar. Requirements: Webhook from Chronicle/Datadog/CloudFlare + optional application events for ownership context. We never require production database access.

Do I need Chronicle or a specific SIEM?

No. We integrate with any SIEM that can send webhooks (Splunk, Datadog, CloudFlare, Sentinel, QRadar). Chronicle is optional but provides rich enrichment.

What about PII and customer data?

All PII is scrubbed locally (Presidio + regex) before any external API call. Customer emails, IPs, credit cards never reach the LLM. Fully GDPR/PCI-DSS compliant.

Can I run this on-premise or in my VPC?

Yes. Docker/K8s deployment. All sensitive data stays in your infrastructure. Only scrubbed, anonymized data hits external AI APIs.

How much does this cost vs. hiring a SOC team?

SOC team: $450K-$750K/year (3-5 analysts). Us: $30K-$60K/year depending on alert volume. Save $400K+/year.

What if I already have some security tools?

We layer on top. Your existing SIEM, EDR, CloudFlare, Datadog all stay. We just validate alerts before they reach your team.

Do you replace my security team?

No. We're the first line of defense. High-confidence alerts we handle automatically. Medium-confidence alerts go to your team with context. You make final decisions on complex scenarios.
